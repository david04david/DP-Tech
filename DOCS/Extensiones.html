Proyecto GITHUB cooperativo



Guia para Usar Extensiones



Azure Repos


Funciones


  • Abra cualquier repositorio de Azure directamente desde Azure Repos; No se requiere clonación ni repositorio local.

  • Copie y pegue vínculos directamente desde Azure Repos para abrir el repositorio directamente en VS Code.


GitHub Codespaces


Funciones


Codespaces funciona como un Visual Studio Code incrustado dentro de GitHub, dándonos la posibilidad de editar, depurar, compilar e incluso depurar nuestro código. Codespaces puede utilizarse de forma natural, es decir, en el navegador, porque sólo se basa en tecnologías en línea.

GitHub Pull Requests/Issues

Funciones


  • Autenticar y conectar VS Code a GitHub y GitHub Enterprise.

  • Acciones de código para crear problemas a partir de comentarios

  • Integración de terminal que permite que la interfaz de usuario y las CLI coexistan.


GitHub Repositories


Funciones

  • Puedes utilizar propuestas para recolectar la retroalimentación de los usuarios.

  • Puedes utilizar los GitHub Discussions para hacer y responder preguntas, compartir información

  • Puedes utilizar las solicitudes de cambios para proponer cambios a un repositorio.


Remote Repositories


Funciones


  • La extensión Repositorios remotos se integra con las extensiones Repositorios de GitHub y Repositorios de Azure.

  • permite examinar, buscar, editar y confirmar rápidamente repositorios.
