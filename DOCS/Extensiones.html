Proyecto GITHUB cooperativo




Funciones


  • Abra cualquier repositorio de Azure directamente desde Azure Repos; No se requiere clonación ni repositorio local.

  • Copie y pegue vínculos directamente desde Azure Repos para abrir el repositorio directamente en VS Code.

  • Los repositorios siempre están abiertos a la última versión.

Tag


Tag


Los Tag permiten controlar el estado de un repositorio dando información a otros usuarios de en qué versión se encuentra actualmente el proyecto. Esta acción es conocida como “Tagging” y es bastante importante a la hora de gestionar la vida de un proyecto. Crear un nuevo proyecto a partir de otro (Fork).
